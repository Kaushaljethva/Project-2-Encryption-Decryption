Project 2: Basic Encryption & Decryption

Description:-
This project implements a Caesar Cipher in Python to perform text encryption and decryption. The program shifts each alphabet character by a fixed value and can restore the original text through decryption.

Features:-
Encrypt text using Caesar Cipher
Decrypt encrypted text back to original text
Handles uppercase and lowercase letters
Preserves spaces and special characters
Uses Python's ord() and chr() functions

How It Works:-
1. User enters a text message.
2. The program shifts each letter by a fixed value (3).
3. The encrypted text is displayed.
4. The encrypted text is decrypted back to the original text.

Example:-
Input:Hello Kaushal
Encrypted Text:Khoor Ndxvkdo
Decrypted Text:Hello Kaushal
